NOTES:
- We detected a multi-step ClickFix attack using multiple deceptive techniques including brand squatting, clipboard decoy, multi-stage executable payloads disguising as logs or images.
- Appearing similar to the legitimate payment platform called lirnunex.com, the threat actor registered the domain lirunex[.]tech a few days ago and launched a highly evasive ClickFix attack with multi-stage payloads eventually leading to the delivery of a cross-platform RAT disguised as image files.
- When a victim pastes what they believe is an SSL certificate path, the threat actor silently replaces it with malicious clipboard paste that uses “conhost --headless” to silently execute the first step acting as a two-stage dropper of the RAT. 
- The website is cloaked for non-targets by showing the Swagger UI clone of the legitimate Lirunex payment gateway API. The cloaking logic is controlled at the server side.

DETAILS:
- Multiple deceptions are used to launch the attack: 
– Lure page domain is a TLD squat of the legitimate trading platform lirunex.com and uses .tech TLD and api-eu3 subdomain to further mimic a fintech company.
– Clipboard content is pasted while the victim is asked to select the SSL certificate
– Clipboard paste has the decoy suffix lirunex_Api_EU3.pem after 250 spaces
– The clipboard script uses conhost.ext –headless to execute the command silently
– The first malicious payload is hidden in the remote file https://colafunfacts[.]net/log 
– The second malicious payload is disguised as a Firefox binary: https://colafunfacts[.]net/firefox.zip
– Python executables inside firefox.zip are named as image files (mev.png, USO.ico, USO.png)
- The full capability set of the RAT includes browser-credential stealing, remote shell, live screen streaming, keylogger, file manager, and SSH tunneling. The RAT persists across reboots on cross platforms. It is also AV/EDR aware and has several anti-forensics mechanisms. Successive payloads are downloaded from the main C2 hxxp[:]//104.207.131[.]216[:]16443
- We also detected the same clipboard injection in additional sites including similar financial themed sites aromi-fr[.]shop, aromi-fr[.]com, and api-eu3.convmasters[.]com
