2026-04-21 (TUESDAY): LummaStealer Delivered Via Malicious Search Results

AUTHOR(S):

- Jonny Westfall, Alex Payne

NOTES:

- Initial Access via SEO Poisoning: The attack chain begins by targeting users searching for YubiKey Manager, delivering a malicious ISO file.

- Complex Execution Chain: The infection utilizes DLL sideloading (PYTHON311.DLL via YUBICO_SETUP.EXE) and defense evasion via PowerShell to add Windows Defender exclusions before executing the primary loader.
- Obfuscated AutoIt Loader: A heavily obfuscated AutoIt script (masquerading as Health.exe) uses RC4 encryption and LZNT1 decompression to unpack the Lumma Stealer payload.
-In-Memory Execution: The malware employs Process Hollowing (RunPE) to inject the C++/Qt Lumma Stealer payload directly into memory, evading traditional file-based detection.
- Advanced Privilege Escalation: The malware duplicates winlogon tokens to gain SYSTEM privileges, allowing it to bypass and decrypt "app-bound" encrypted keys.
-Extensive Data Theft Scope: The stealer aggressively targets 24 Chromium-based browsers, Firefox, multiple FTP/SSH clients (FileZilla, WinSCP, PuTTY), and KeePass password manager configurations.
- Robust Backdoors: Beyond simple exfiltration, it establishes an RDP backdoor (rdpwrap.dll), a reverse tunnel via Plink, and force-installs malicious Chrome extensions.
- Anti-Analysis & Cleanup: The malware checks for known sandbox hostnames to evade analysis and utilizes a slick RunOnce registry modification (wextract_cleanup0) to delete temporary creation files upon user logon, hindering forensic recovery.
- LummaStealer Attribution: Attribution to the LummaStealer malware family is confirmed through several distinct characteristics and tactical overlaps observed in this sample.
-- The reliance on AutoIt scripts for payload delivery, combined with the use of .TOP domains for infrastructure, serve as strong initial indicators. 
-- Deeper analysis of the AutoIt payload reveals it performs environmental checks for specific sandbox hostnames historically associated with Lumma campaigns—specifically checking for test22, NfZtFbPfH, and ELICZ. 
-- The payload is tied to the creation of a highly specific scheduled task named NeuraLogix, a recognized hallmark that firmly roots this execution chain to LummaStealer.
