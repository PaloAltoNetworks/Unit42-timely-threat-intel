2025-08-19 (TUESDAY): CHROME EXTENSIONS FOR CUSTOMIZED CURSORS LEAD TO ADWARE/PUP

AUTHORS:

- Shresta Bellary Seetharam, Zhanhao Chen, Nabeel Mohamed, Qinge Xie, Shehroze Farooqi, Fang Liu

NOTES:

- We found 100+ Chrome extensions that customize cursors in the browser window. 
- These extensions have appeared in the Chrome Web Store during the last 60 days and have 1000+ installs.
- These extensions appear benign, offering themed cursors (cartoon, anime, Pokémon, etc.) to entice users. 
- Each extension comes pre-packaged with a few cursors but includes a "Download more cursors" button for additional cursors.
- Clicking this button redirects to pages that deliver adware or potentially unwanted programs (PUPs), using domains such as:
  -- owhit[.]com
  -- pixatab[.]xyz
- Many of these extensions share identical source code, 
- These extensions often include permissive features such as MutationObserver and differ only in the themed cursors.

RECOMMENDATIONS FOR USERS:

- Avoid installing cursor-modifying extensions from unverified sources. 

RECOMMENDATIONS FOR ADMINISTRATORS: 

- Enforce policies that define allowed extensions and block installation of potentially harmful extensions in enterprise browsers.

- owhit[.]com
- pixatab[.]xyz
