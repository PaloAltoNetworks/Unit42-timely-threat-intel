2025-09-23 (TUESDAY): ONGOING PHISHING CAMPAIGN USING BROWSER-IN-THE-MIDDLE (BITM) TECHNIQUE

AUTHORS:

- Beliz Kaleli, Alex Starov, Lucas Hu, Shawn Shuang, Nabeel Mohamed, Bradley Duncan

NOTES:

- We're tracking a phishing campaign active as early as April 2025 that uses the browser-in-the-middle (BitM) technique.
- This campaign is impersonating Meta and Facebook for its phishing pages.
- We have found copyright infringement notices used as phishing emails with the initial URLs for this activity.
- This campaign remains active, as we continue to find new domains and URLs. 

CURRENT CHAIN OF EVENTS:

- email --> URL from message --> redirect URL --> Phishing page using BitM

- Note: We found variations of this chain
  For example, at least one phishing email from 2025-04-19 linked directly to the BitM page.

DETAILS:

- BitM is a technique that displays a fake browser window within a potential victim's legitimate browser window.
- These fake browser windows display a legitimate URL, but the content is used to steal login credentials.
- In this particular campaign, the phishing emails contain links that lead to a Meta-themed fake CAPTCHA page.
- Clicking the "I'm not a robot" box on the fake CAPTCHA page will display the fake browser window.
- The fake browser window impersonates a Facebook login page and displays a legitimate Facebook login URL.
- This fake browser window is used to collect login credentials for Facebook accounts.
- This campaign occasionally changes the domains of its URLs for the BitM landing pages.
- Therefore, the same link from a phishing email may not land on the same BitM page from one week to the next.
- Pivoting on current information shows indicators of this campaign as early as April 2025.
