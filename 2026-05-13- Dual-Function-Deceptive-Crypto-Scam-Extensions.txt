2026-05-13 [WEDNESDAY]: Dual-Function/Deceptive Crypto Scam Extensions

AUTHOR: Nabeel Mohamed, Shresta Bellary Seetharam, Qinge Xie, Fang Liu, and Oleksii Alex Starov 

NOTES:

- We observe a shared evasion philosophy across all malicious extensions where catch {} always defaults to the clean reviewer-safe path, every extension silently falls back to benign behavior if the C2 is unreachable, ensuring no malicious activity is visible during Chrome Web Store review sandbox execution.
- The use of Unicode BIDI spoofing makes malicious extensions indistinguishable from Ledger Live and Braavos — \u202E (RIGHT-TO-LEFT OVERRIDE) reverses rendered text in the Chrome Web Store so it displays as Ledger and a reversed string displays as Braavos extension: Bitcoin & Starknet Wallet, defeating any string-matching scanner.
- All C2 domains returned zero or 1 VirusTotal detections showing that reputation based analysis fails to spot them. 
- The BSC drainer (Token Rewards Claim) sends eth_sendTransaction with attacker-controlled parameters, including contract address, nonce, deadline, and signature all come from api[.]openaitoken[.]org; the victim's wallet shows a legitimate-looking approval dialog with no indication the transaction is a drain.
 

DETAILS:

- Ledger Live v2.0.1 (gkcamdgljboohmjlfmogjnbckhcbpdhi) appears as a CoinGecko crypto portfolio tracker; when the actor's remote kill-switch (academise[.]org/ ext-mode.php) returns "dirty", it opens a full-screen tab loading a phishing iframe from academise[.]org/edge/ with clipboard read/write access to capture Ledger seed phrases and private keys. The extension name uses Unicode RTL override (\u202E) to display as Ledger Live in the Chrome Web Store.
- Solana Monitor v2.0.2 (ggijepplmdjopbidkeoeheohojgiclma) appears as a Solana network dashboard; when kill-switch at dylimu[.]com/pntm/mode.php returns "dirty", swaps the popup to a pixel-perfect bundled Phantom wallet replica that exfiltrates seed phrases and private keys via HTTP POST to dylimu[.]com/pntm/relay.php before showing a cosmetic password screen. Zero permissions declared.
- OmniFlow Sidebar / Blockchain Wallet v1.1.2 (pfgpfmdiepmhhhkpnciogjhccppbcfhk) appears to English users as a fully functional productivity sidebar (notes, drawing, image editing, 60+ language translation); appears to all 43 non-English locales as "Blockchain Wallet: Buy BTC" with zero crypto functionality. Non-English users who click the icon are silently redirected to http://swifnotees[.]online/ with no pop-up, no warning, and no consent.
- Crypto Dashboard & Calculator v1.0 (mgbkajajjhgmhchoajmomhcdmlndkofc) appears as a crypto price dashboard and calculator tool; acts as an earlier-stage browser hijacker redirecting non-English users to timeanddate[.]homes The same publisher (pitshopping1@gmail[.]com) as OmniFlow.
- Token Rewards Claim v2.1.0 (koecfhecfkmjkjplbmbclgacdclnnpei) appears as a ChatGPT closed-beta token rewards program; injects a fake "领取 $Token" (Claim $Token) navigation link into the ChatGPT sidebar and displays a live-accruing fake balance counter. When the user connects their MetaMask or OKX wallet to "claim" rewards, fires eth_sendTransaction on BSC with all parameters, including contract address, amount, nonce, deadline, signature, supplied by the attacker's C2 server (api[.]openaitoken[.]org), draining funds from the victim's wallet.
- ZAP Wallet v1.2.1 (cdiohdbijdajffgccjmbblbikpnnnkeg) appears as an AI-powered, dApp-compatible Solana wallet; ships a real BIP39 wallet generator (src/walletGenerator.js) that is never called, and a fake deployed generator (popup/background-bundle.js) using a 56-word wordlist, Math.random(), and "ZAP" + hex address encoding, producing addresses that are not valid Solana addresses and cannot receive SOL. All three dApp signing methods throw "Not yet implemented" despite advertising dApp compatibility. It injects window.solana override into every website the user visits.
- Braavos spoof v1.1.5 (clehpjldcdennhphcfihphbeoihllbki) appears as Braavos extension : Bitcoin & Starknet Wallet via Unicode RTL override in the extension name, with zero-width spaces (\u200B) throughout the description to defeat string-matching scanners; shows a fake "SILVER VAULT / AES-256 / SECURE" key generator UI when the C2 is offline. When the Cloudflare Workers C2 (encryptedapikeyscanner[.]slmgames2020.workers[.]dev) is reachable, wipes the popup DOM and injects a full-viewport phishing iframe serving the attacker's credential-harvest page where no permissions declared, IP-level  blocking ineffective due to Cloudflare edge infrastructure.
