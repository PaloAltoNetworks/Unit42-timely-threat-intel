2026-01-22 (THURSDAY): ATTACK CHAIN TARGETS USERS SEARCHING FOR LEGITIMATE TOOLS

AUTHORS:

- Shelly Markus, Manbendra Satpathy

NOTES:

- We identified an attack chain leveraging SEO poisoning to target users searching for legitimate tools. 
- Attackers have abused GitHub to create repositories hosting malicious ZIP archives.
- These ZIP archives masquerade as legitimate programs and contain a malicious batch (.bat) file.
- Executing the .bat file exfiltrates host details to a C2 server via cURL.
- This is followed by encoded PowerShell script for system reconnaissance.
- In at least one occasion, a C2 server deployed a TeamViewer binary as a remote admin tool.

DETAILS:

- Malicious ZIP archives from this attack chain have appeared in the wild as recently as November 2025.
- Examples of SEO poisoning and associated repositories remain active as of January 2026.
- However, the ZIP archives on the associated repositories no longer contain malicious .bat files.
- Instead, these ZIP archives now contain copies of the legitimate files or installers for these tools.
- As a result, we have found no new malware from this attack chain.
- However, existing infrastructure remains, and we will monitor for further examples of the malware.
